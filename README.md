REAL TIME ANOMALY DETECTION SYSTEM WITH GUI  
  
This is a system that performs visual inspection using AI.  
This is a real-time version of cflow-ad (it learns only from normal images).  
Detects abnormalities from images obtained from a web camera.  
Both learning and detection operations can be performed on the GUI.  
Please refer to readme_en.txt for details.  
  
---
  
GUIによるリアルタイム異常検知システム  
  
AIで外観検査を行うシステムです。  
cflow-adのリアルタイム版となります（正常画像のみで学習します）。  
WEBカメラより取得した画像から、異常を検出します。  
学習と検出の操作は、それぞれGUI上で行う事が出来ます。  
詳細はreadme.txtを参照して下さい。
